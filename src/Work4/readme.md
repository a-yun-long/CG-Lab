# Taichi 实时光线追踪渲染器 (Taichi Ray Tracer)

## 项目简介

本项目是一个基于 Python 和 Taichi 框架实现的  **GPU 实时光线追踪渲染器** 。抛弃了传统的多边形网格（Mesh）与光栅化管线，本项目完全基于隐式几何方程构建三维物理世界。通过纯数学解析与蒙特卡洛积分，实现了光线的物理弹射、硬阴影遮蔽、镜面反射以及真实的玻璃折射效果。

## ✨ 核心特性

本项目不仅包含基础的光追特性，还实现了多个进阶图形学算法：

* **隐式几何相交：** 不依赖外部模型，完全通过代数方程在 Shader (Taichi Kernel) 中动态求交（无限大棋盘格平面、完美球体）。
* **迭代式路径追踪：** 考虑到 GPU 架构对递归（Recursion）的限制，将光线追踪深度循环展开（Iterative Ray Bouncing），大幅提升并行渲染性能。
* **硬阴影与自相交修复：** 通过向点光源发射暗影射线（Shadow Ray）实现精确阴影。引入沿法线方向的微量偏移（**$\epsilon = 10^{-4}$**），完美消除了浮点数精度导致的“Shadow Acne”（自相交黑斑）Bug。
* **物理级玻璃折射 (选做拓展)：** 引入斯涅尔定律（Snell's Law）。当光线在不同折射率介质中穿梭时，不仅计算精确的偏折角，还处理了光线从介质内部射出时的**全反射 (Total Internal Reflection, TIR)** 现象。
* **多重采样抗锯齿 (MSAA 选做拓展)：** 采用子像素级随机采样（Sub-pixel Jittering），在一个像素内发射多条射线并求均值，将高频锯齿转换为平滑的低频色彩过渡。

## 🛠️ 运行环境与依赖

运行本项目无需复杂的 C++ 环境配置，仅需标准的 Python 环境及 Taichi 库：

* Python 3.7+
* Taichi 1.x+
* *(无需额外安装第三方 UI 库，已使用 Taichi 内置的 GUI 系统)*

**安装依赖：**

**Bash**

```
pip install taichi
```

## 🚀 快速开始

克隆本仓库到本地后，直接运行主脚本即可开启实时渲染窗口：

**Bash**

```
git clone <你的仓库链接>
cd <仓库目录>
python main.py
```

## 📐 代码结构与技术要点

* `scene_intersect()`: 负责场景遍历，返回最近交点距离、材质 ID 及法线信息。
* `render()`: Taichi GPU Kernel 核心入口，负责并行处理像素、执行 MSAA 采样并管理 `throughput` 衰减。
* **法线翻转逻辑** ：在玻璃材质渲染中，通过判断入射光与法线的点积符号，动态翻转出射法线，确保光线穿透介质时的数学逻辑自洽。
