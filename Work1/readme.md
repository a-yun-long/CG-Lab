# 实验二：从平面三角形到 3D 彩色立方体

本项目完整实现了计算机图形学中的 **MVP (Model-View-Projection)** 变换流程，并利用 **Taichi** 框架进行了实时渲染演示。

---

## 阶段一：基础任务 - 平面三角形 (`lab.py`)
在 3D 空间中定义了一个平面三角形，通过矩阵运算将其投影至 2D 屏幕。
- **核心逻辑**：实现了基本的模型、视图与透视投影矩阵。
- **运行演示**：
  ![平面三角形演示](./triangle.gif.gif)

---

## 阶段二：进阶挑战 - 3D 彩色立方体 (`lab1.py`)
在基础 MVP 逻辑之上，构建了完整的 3D 几何体结构。
- **空间建模**：定义了 8 个顶点与 12 条连接边。
- **色彩区分**：
  - <font color="red">**红色**</font>：后表面线框
  - <font color="green">**绿色**</font>：前表面线框
  - <font color="blue">**蓝色**</font>：侧向连接边
- **交互功能**：支持通过键盘实时控制旋转及重置状态。
- **运行演示**：
  ![彩色立方体演示](./cube.gif.gif)

---

## 🛠️ 环境要求
- **Python**: 3.12 (Conda 环境: `cg_env`)
- **框架**: Taichi Lang

## ⌨️ 操作指南
| 按键 | 功能说明 |
| :--- | :--- |
| **A / D** | 控制模型左右旋转 |
| **T** | 将旋转角度重置为 0 |
| **Esc** | 退出程序窗口 |
